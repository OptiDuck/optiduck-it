# OptiDuck IT E-Mail-Signatur

## HTML-Signatur

Die Datei `optiduck-signature.html` enthält die Signatur mit Logo.

Das Logo wird von dieser URL geladen:

```text
https://www.optiduck-it.de/optiduck-email-signature-logo.png
```

Die Datei `public/optiduck-email-signature-logo.png` muss online deployed sein, bevor die Signatur in Mailprogrammen zuverlässig angezeigt wird.

## Text-Signatur

Die Datei `optiduck-signature.txt` ist die schlichte Textversion ohne Logo.

## Hinweis

Viele Mailprogramme blockieren Bilder beim ersten Öffnen einer E-Mail. Das ist normal. Die Signatur bleibt trotzdem lesbar, weil Name, Kontakt und Website zusätzlich als Text enthalten sind.
