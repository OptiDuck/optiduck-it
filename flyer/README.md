# OptiDuck IT Flyer

## Dateien

- `output/flyer/optiduck-it-flyer-a5-mit-3mm-beschnitt.pdf`: Druckereiversion mit 3 mm Beschnittzugabe.
- `output/flyer/optiduck-it-flyer-a5-standard.pdf`: DIN-A5-Version für Bürodruck oder digitales Teilen.
- `output/flyer/optiduck-it-flyer-vorderseite.png`: Vorschau Vorderseite.
- `output/flyer/optiduck-it-flyer-rueckseite.png`: Vorschau Rückseite.

## Vor dem Druck prüfen

- Der QR-Code verweist auf `https://www.optiduck-it.de`.
- Die Domain sollte vor dem Verteilen erreichbar sein.
- Die E-Mail-Adresse `info@optiduck-it.de` ist als aktives Strato-Postfach eingerichtet.
- Für eine Druckerei die Datei mit 3 mm Beschnitt verwenden.
